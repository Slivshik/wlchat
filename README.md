## WL CHAT Android Client

A [Delta Chat](https://delta.chat/) client for Android. Fork of ArcaneChat and Delta Chat.

A chat app that uses email as a relay. Your own email server is the default; ready-made chatmail servers are optional.

# Credits

This app is based on the [ArcaneChat](https://arcanechat.me) client, which is based on the [official Delta Chat client](https://github.com/deltachat/deltachat-android).
